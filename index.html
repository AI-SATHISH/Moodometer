📲 Daily check-in reminder

Turn on notifications so this device gets a daily nudge to check in. This is a broadcast reminder — since check-ins are anonymous, everyone gets the same message rather than a personal one.

🔒 Privacy & data

Check-ins are anonymous by design — no name, login, or email is collected. A random ID generated on each device lets HR spot repeated patterns without knowing who's behind them. Employees can optionally ask HR to reach out directly; only then is contact info shared. All data is stored in browser local storage on this device only — nothing is sent to an external server in this version.
